AG BuyCars

UI Design and Web Development for a second-hand car dealer

UI Design

  • Research and got inspirations from other auto-dealers.
  • Created a simple design, allowing the admin and owner to easily navigate the website and manage their car inventory.
  • Used Figma community assets to access high quality car pictures that I could use for the design of the website.

Web Development

  • Generated the Figma design using Next.js.
  • Developed an authentication system with Firebase Auth.
  • Used SQLite to manage the car inventory as a database.
  • Enabled a PDF report download for streamlined tracking.
